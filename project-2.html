Overview

Fantasy Sports Application : It is a project where it creates a platform for the cricket fans to use their skills and make their passion into earning. As cricket is not just a sport in India, it's a religion. From Tests to T20s, you can join contests for all formats of international cricket on FantasyField. Apart from ICC matches and tournaments, users can also play online cricket games from various cricket leagues like the IPL, Bigbash - followed and played across the world. In this project we Designed and solved the user centric problem. And enhanced The experience of the user and made the application fun. So, that user can easily interact without any effort.

Challenge

As there are many existiing applications, their experience was quite Lengthy. User are loosing interest in the fantsy sports. In current application user is going through the unnecessery path which he already passed through it. The UI of current application was also not fun that is also a reason for loosing interest in the game.

-The main challenge was to get something new in this application, As there are many compititors in market so its was an very challing task to get something new.

Solution

An short and crisp flow, so that user can eaily navigate where he wants to go without getting distracted. And making the UI attractive to engage the user in the application to spend time in it.

-To make something different from others we introduced the auction in this version, where user can actually engage with the game by using stratergys and skills. Here user can actually bid for the best players with the other users and make their best team and win lot of money. We also introduced the sponsership auction leagues into it bussines growth.

User Flows & Sitemap

Userflow are flow chart diagrams that explains the different process involved in accomplishing a task from your MVP. It lays out steps & actions in particular order and sequence that the user needs to undertake in order to get the end of the task. Since its an fantasy sports application, the most important user flow is for a user to click on the match and join leagues and watch the rank. There are several userflows that explain how other tasks are to be done. Besides the sitemap explains the skeleton on what the site is built upon.

Wireframing & Prototyping

Wireframes sketching is the first step where you translate written matter into its visual layout. They are low fidelity in nature and does not contain an actual copy. They help you to decide the visual layout by planning appropriate UI elements and sizing them accordingly.

Visuals Designing

This section marks the beginning of UI deliverable. it’s the most exciting part of the design process.
